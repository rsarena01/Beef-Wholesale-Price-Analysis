### US Wholesale Beef Price Analysis

## Overview

This repository contains an econometric data and modeling pipeline that explores the impact of major supply chain factors—specifically imports, exports, and total production—on real wholesale beef prices. The analysis standardizes historical marketplace dynamics into an explanatory regression framework to evaluate how global trade and domestic production shape wholesale costs.

The dataset was constructed using publicly available agricultural and economic data sources:

* **USDA Economic Research Service:** Livestock and Meat Domestic Data (Last Updated: May 27, 2026)  
    [https://www.ers.usda.gov/data-products/livestock-and-meat-domestic-data](https://www.ers.usda.gov/data-products/livestock-and-meat-domestic-data)
* **Federal Reserve Bank of St. Louis (FRED):** Producer Price Index by Commodity: All Commodities (PPIACO)  
    [https://fred.stlouisfed.org/series/PPIACO](https://fred.stlouisfed.org/series/PPIACO)

## File Descriptions and Workflow Integrations

### Data Directories
* **`data/raw/`**: Contains the raw, multi-source data files ingested by the cleaning script.
* **`data/processed/beef_market_clean_dataset.csv`**: The primary data asset generated by the cleaning pipeline. It consists of 26 annual observations (spanning 2000 to 2025) across 19 features. All continuous metrics are cast as `float64`, with the temporal index year cast as `int64`. The dataset contains zero missing or null values.
* **`data/results/experimental_results.csv`**: A performance matrix archiving the $R^2$, adjusted $R^2$, information criteria (AIC/BIC), hypothesis testing, and diagnostic baselines generated across all experimental regression variations.

### Source Code & Notebooks
* **`import_and_clean.py`**: The entry point of the pipeline. Running this script automatically loads data from `data/raw/`, resolves headers and metadata artifacts, standardizes formatting, and maps variables to intuitive feature names. It aggregates the final fields into a cohesive table and outputs the clean file to the processed directory.
* **`eda.ipynb`**: Handles feature exploration, statistical profiling, and baseline experimental model architecture. This notebook tracks historical price distributions, identifies annual variance trends, maps structural market cycles, and executes several experimental regressions. The summary metrics from these iterative modeling runs are automatically exported to the results directory.
* **`supply_factors_per_capita_analysis.ipynb`**: Isolates and runs the production-ready model configuration. The pipeline evaluates the final specification for multicollinearity, autocorrelation, and homoskedasticity. It concludes with an economic interpretation of the coefficients and outlines key areas for future structural research.

## Methodology & Analytical Scope

### 1. Target Feature Selection & Inflation Adjustment
Select wholesale beef prices (`select_price`) were isolated as the primary target metric for this analysis because they offer the most stable, consistent variation across the 2000–2025 horizon. To remove macro inflationary distortions and isolate real economic variations, prices were deflated to **Real 2025 Dollars per Hundredweight ($/cwt$)** using the FRED Producer Price Index (`PPIACO`).

### 2. Analytical Boundaries
The regression pipeline prioritizes a focused, explanatory framework centering on the three pillars of marketplace supply: **Imports**, **Exports**, and **Total Production**. Secondary equilibrium metrics—such as `begin_stock` and `total_disappearance`—are omitted from the immediate modeling scope.

### 3. Model Validation & Classical Assumptions
The production-ready iteration evaluated in `supply_factors_per_capita_analysis.ipynb` undergoes statistical verification to ensure BLUE (Best Linear Unbiased Estimator) properties:
* **Multicollinerity:** Evaluated using Variance Inflation Factors (VIF) to ensure independent variables do not exhibit destabilizing cross-correlations (Target: $\text{VIF} < 5.0$).
* **Autocorrelation:** Evaluated via the Durbin-Watson statistic to monitor sequential dependence in time-series residuals (Target: $d \approx 2.0$).
* **Homoskedasticity:** Evaluated using variance diagnostic plots and residual testing to verify constant variance of error terms across predicted values.

# How to Run the Project

## 1. Install Dependencies

Install required Python packages:

``` bash
pip install pandas numpy matplotlib seaborn statsmodels openpyxl notebook
```

------------------------------------------------------------------------

## 2. Ingest and Generate the Clean Dataset

Run:

``` bash
python import_and_clean.py
```

* **Input Dependencies:** `data/raw/MeatSDFull.xlsx`, `data/raw/WholesalePrices.xlsx`, `data/raw/PPIACO.csv`
* **Output:** Writes to `data/processed/beef_market_cleaned.csv`

------------------------------------------------------------------------

## 3. Exploratory Modeling & Feature Prototyping

Open the exploratory workspace to review descriptive data profiles and raw iterations:

``` bash
jupyter notebook eda.ipynb
```

* **Action:** Evaluates baseline distributions, visualizes market cycles, runs exploratory multi-feature OLS trials, and appends metric logs into `data/results/experimental_results.csv`.

------------------------------------------------------------------------
# 4. Execute Final Per-Capita Verification Pipeline
Run the production notebook to validate the final model against classical linear regression assumptions:

``` bash
jupyter notebook supply_factors_per_capita_analysis.ipynb
```

* **Action**: Evaluates per-capita scaled trade drivers, checks VIF and residual stability, and provides economic synthesis on the parameter estimates.
